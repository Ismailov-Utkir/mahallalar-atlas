# Маҳаллалар Тадбирлар Атласи

Ўзбекистон Маҳаллалари Уюшмаси учун тадбирлар атласи ва админ панел.

## Саҳифалар

- `/` — жамоат харитаси (Leaflet асосида)
- `/admin` — админ панел (тадбир қўшиш, рўйхат, статистика)

## Технологиялар

- HTML / CSS / JavaScript (статик)
- Leaflet.js — интерактив харита
- Google Fonts (PT Serif, PT Sans)

## Локал ишга тушириш

Файлларни брауз ерда очиш кифоя — backend йўқ.

```
start index.html
```

## Деплой

Vercel'га ўрнатилган. `main` тармоғига push қилинса, автоматик янгиланади.
